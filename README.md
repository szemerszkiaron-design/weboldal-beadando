Weboldal elérhetősége: 

Téma leírása:
A weboldal az okosvilágítás és a LED technológia előnyeit mutatja be. Tartalmaz egy termékbemutatót, egy megtérülés kalkulátort és egy kapcsolatfelvételi űrlapot.

JavaScript kód leírása:
A saját JavaScript kódom a 'js/script.js' fájlban található.
A kód két fő funkciót lát el:
1. Kapcsolat űrlap validáció: Ellenőrzi, hogy a felhasználó minden kötelező mezőt (név, email, terület, típus) helyesen töltött-e ki.
2. Megtérülés kalkulátor: A felhasználó által megadott adatok alapján kiszámolja az éves megtakarítást, és dinamikusan frissíti az eredményt.

Felhasznált betűtípus:
Google Fonts - 'Poppins' (Sans-serif)
Importálva a css/style.css fájlban.

Külső források:
- A kód vázának és a validációs logikának a megértéséhez mesterséges intelligencia (Gemini) segítségét vettem igénybe, de a kódot átnéztem és értelmeztem.
